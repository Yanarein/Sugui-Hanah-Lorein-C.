# Shopping Cart System (Part 2)

# Project Overview

The Shopping Cart System is a C# console-based application that simulates a Point of Sale (POS) and inventory management system. It allows users to browse products, manage a shopping cart, process checkout transactions, and track order history.

This project demonstrates Object-Oriented Programming (OOP), collections, input validation, and system logic design.

# Features

### Product Management

* View all products
* Search products by name
* Filter products by category
* Displays product details (ID, price, stock, category)

# Shopping Cart

* Add products to cart
* Update quantity
* Remove items
* Clear cart

# Checkout System

* Automatic receipt generation
* Receipt number and date tracking
* Discount system (10% for ₱5000 and above)
* Payment validation
* Change computation

# Inventory Management

* Stock decreases when items are added to cart
* Stock restores when items are removed
* Low stock alert (≤ 5 items)

# Order History

* Stores completed transactions
* Displays receipt number, total amount, and timestamp

## Technologies Used

* Visual Studio (C# .NET Console Application)
* Object-Oriented Programming (OOP)
* Collections (List, Dictionary)
* Input validation techniques
